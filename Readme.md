# ✈️ Sky-Link Airlines Reservation System

A comprehensive airline reservation system built with C# .NET 6.0 and MongoDB. Manage domestic and international flight bookings, loyalty programs, real-time check-in, and more!

## 🚀 Features

### Customer Features
- **User Authentication** - Register/Login with secure password encryption
- **Domestic & International Booking** - Book flights with seat selection
- **Loyalty Program** - Earn miles on every booking, redeem for discounts
- **Web Check-in** - Online check-in 48 hours before departure
- **Promo Codes** - Apply discount codes for special offers
- **PNR Search** - Check booking status anytime
- **Cancel Booking** - Cancel with refund calculation
- **Feedback System** - Rate and review your experience
- **Email Notifications** - Receive booking confirmations and updates

### Admin Features
- **Flight Schedule Management** - CRUD operations for flights
- **Booking Management** - View, update, cancel all bookings
- **Promotion Management** - Create and manage discount codes
- **User Management** - View registered users
- **Loyalty Program Management** - View top members
- **Real-time Flight Status** - Update flight statuses
- **Export Data** - Export reports to CSV
- **System Statistics** - Comprehensive analytics dashboard

## 🛠️ Technologies Used

- **Backend:** C# .NET 6.0
- **Database:** MongoDB
- **Email Service:** SMTP (Gmail)
- **Container:** Docker & Docker Compose
- **Version Control:** Git

## 📋 Prerequisites

- .NET 6.0 SDK or later
- MongoDB (local or Docker)
- Git (for cloning)
